## Overview

LDAP (Lightweight Directory Access Protocol) is a protocol for accessing and managing distributed directory information services. It is the backbone of **Microsoft Active Directory** and widely used in enterprise environments for authentication, authorization, and user/group management. It listens on **TCP port 389** (plaintext/STARTTLS) and **TCP port 636** (LDAPS — implicit TLS). LDAP is one of the most critical targets in penetration tests because it stores the **entire directory structure** including users, groups, computers, organizational units, Group Policy Objects, service accounts, and often exposes password hashes, Kerberos data, and ACL misconfigurations.

::card-group
  ::card
  ---
  title: Default Ports
  icon: i-lucide-network
  ---
  `389/TCP` (LDAP plaintext / STARTTLS), `636/TCP` (LDAPS implicit TLS), `3268/TCP` (Global Catalog), `3269/TCP` (Global Catalog SSL)
  ::

  ::card
  ---
  title: Protocol
  icon: i-lucide-cable
  ---
  Binary ASN.1/BER encoded protocol over TCP. Operations include Bind, Search, Add, Modify, Delete, Compare, Extended. Search filters use prefix notation `(& (objectClass=user)(cn=admin))`.
  ::

  ::card
  ---
  title: Authentication
  icon: i-lucide-lock
  ---
  **Simple Bind** (plaintext username/password), **SASL** (GSSAPI/Kerberos, DIGEST-MD5, NTLM, EXTERNAL), **Anonymous Bind** (no credentials — often enabled by default). AD supports NTLM and Kerberos via SASL.

## LDAP Fundamentals

### Directory Structure

::collapsible

| Term | Description | Example |
| ---- | ----------- | ------- |
| **DN** | Distinguished Name — full path to an object | `CN=John Smith,OU=Users,DC=corp,DC=local` |
| **DC** | Domain Component — domain parts | `DC=corp,DC=local` (corp.local) |
| **OU** | Organizational Unit — container | `OU=IT,DC=corp,DC=local` |
| **CN** | Common Name — object name | `CN=Administrator` |
| **RDN** | Relative Distinguished Name | `CN=John Smith` |
| **Base DN** | Root of the search tree | `DC=corp,DC=local` |
| **objectClass** | Schema class defining object type | `user`, `group`, `computer`, `organizationalUnit` |
| **sAMAccountName** | Pre-Windows 2000 logon name (AD) | `jsmith` |
| **userPrincipalName** | UPN logon format (AD) | `jsmith@corp.local` |
| **memberOf** | Group memberships | `CN=Domain Admins,CN=Users,DC=corp,DC=local` |
| **userAccountControl** | Account flags bitmask (AD) | `512` = Normal, `514` = Disabled, `66048` = Password never expires |

::

### Key LDAP Search Filters

::tip
LDAP search filters use prefix (Polish) notation. Master these filters for efficient enumeration during pentests.

```text [ldap-filters.txt]

(objectClass=user)                           All user objects
(objectClass=person)                         All person objects
(objectCategory=person)                      All person objects (AD-optimized)
(&(objectClass=user)(objectCategory=person)) All user accounts (not computers)
(sAMAccountType=805306368)                   Normal user accounts only
(userPrincipalName=*)                        All accounts with UPN

=== ADMIN / PRIVILEGED ACCOUNTS ===
(adminCount=1)                               Accounts with admin privileges
(memberOf=CN=Domain Admins,CN=Users,DC=corp,DC=local)  Domain Admin members
(memberOf=CN=Enterprise Admins,CN=Users,DC=corp,DC=local)  Enterprise Admin members
(memberOf=CN=Administrators,CN=Builtin,DC=corp,DC=local)   Local Administrators

=== DISABLED / LOCKED ACCOUNTS ===
(userAccountControl:1.2.840.113556.1.4.803:=2)    Disabled accounts
(lockoutTime>=1)                                    Locked out accounts
(!(userAccountControl:1.2.840.113556.1.4.803:=2))  Enabled accounts only

=== KERBEROS ATTACKS ===
(&(objectClass=user)(servicePrincipalName=*))      Kerberoastable accounts (SPN set)
(userAccountControl:1.2.840.113556.1.4.803:=4194304) ASREPRoastable (no preauth)
(&(objectClass=user)(msDS-AllowedToDelegateTo=*))  Constrained delegation
(userAccountControl:1.2.840.113556.1.4.803:=524288) Trusted for delegation (unconstrained)

=== PASSWORD POLICY ===
(userAccountControl:1.2.840.113556.1.4.803:=65536)  Password never expires
(pwdLastSet=0)                                       Must change password at next logon
(&(objectClass=user)(!(userAccountControl:1.2.840.113556.1.4.803:=2))(pwdLastSet<=132000000000000000)) Old passwords

=== COMPUTER ACCOUNTS ===
(objectClass=computer)                        All computer objects
(objectCategory=computer)                     All computers (AD-optimized)
(operatingSystem=*Server*)                    Servers
(operatingSystem=*Windows 10*)                Windows 10 workstations
(operatingSystem=*Windows 11*)                Windows 11 workstations

=== GROUPS ===
(objectClass=group)                           All groups
(groupType:1.2.840.113556.1.4.803:=2147483648) Security groups
(groupType:1.2.840.113556.1.4.803:=8)          Universal groups

=== ORGANIZATIONAL UNITS ===
(objectClass=organizationalUnit)              All OUs

=== GPO ===
(objectClass=groupPolicyContainer)            All Group Policy Objects

=== TRUST RELATIONSHIPS ===
(objectClass=trustedDomain)                   Domain trusts

=== SENSITIVE DATA ===
(description=*pass*)                          Descriptions containing 'pass'
(comment=*pass*)                              Comments containing 'pass'
(info=*pass*)                                 Info fields containing 'pass'
```

### Anonymous Bind Enumeration

::warning
Anonymous bind is **enabled by default** on many LDAP implementations including older Active Directory configurations and OpenLDAP. This allows unauthenticated enumeration of the entire directory.

  :::tabs-item{icon="i-lucide-code" label="Python (ldap3)"}
  ```python [ldap_enum.py]
  #!/usr/bin/env python3
  import ldap3
  import sys

  target = sys.argv[1] if len(sys.argv) > 1 else "<target>"
  base_dn = sys.argv[2] if len(sys.argv) > 2 else ""

  server = ldap3.Server(target, get_info=ldap3.ALL)

  # Anonymous bind
  conn = ldap3.Connection(server, auto_bind=True)

  # Get base DN from RootDSE if not provided
  if not base_dn:
      base_dn = server.info.other.get('defaultNamingContext', [''])[0]
      if not base_dn:
          ncs = server.info.naming_contexts
          base_dn = ncs[0] if ncs else ''

  print(f"[*] Target: {target}")
  print(f"[*] Base DN: {base_dn}")
  print(f"[*] Server Info: {server.info}")

  # Enumerate users
  conn.search(base_dn,
      '(&(objectClass=user)(objectCategory=person))',
      attributes=['sAMAccountName', 'userPrincipalName', 'memberOf',
                  'description', 'mail', 'userAccountControl',
                  'pwdLastSet', 'lastLogon', 'adminCount',
                  'servicePrincipalName'])

  print(f"\n[*] Users found: {len(conn.entries)}")
  for entry in conn.entries:
      sam = entry.sAMAccountName.value if entry.sAMAccountName else 'N/A'
      desc = entry.description.value if entry.description else ''
      uac = int(entry.userAccountControl.value) if entry.userAccountControl else 0
      spn = entry.servicePrincipalName.values if entry.servicePrincipalName else []
      admin = entry.adminCount.value if entry.adminCount else 0

      flags = []
      if uac & 2: flags.append('DISABLED')
      if uac & 0x400000: flags.append('NO_PREAUTH')
      if uac & 0x10000: flags.append('PWD_NEVER_EXPIRES')
      if uac & 0x80000: flags.append('TRUSTED_DELEGATION')
      if admin == 1: flags.append('ADMIN')
      if spn: flags.append(f'SPN({len(spn)})')

      flag_str = f" [{', '.join(flags)}]" if flags else ""
      print(f"  {sam}{flag_str}")
      if desc and any(k in desc.lower() for k in ['pass', 'pwd', 'cred', 'key']):
          print(f"    [!] Description: {desc}")

  # Enumerate groups
  conn.search(base_dn, '(objectClass=group)',
      attributes=['cn', 'member', 'description'])
  print(f"\n[*] Groups found: {len(conn.entries)}")
  for entry in conn.entries:
      cn = entry.cn.value
      members = len(entry.member.values) if entry.member else 0
      if cn in ['Domain Admins', 'Enterprise Admins', 'Administrators',
                'Schema Admins', 'Account Operators', 'Backup Operators',
                'Server Operators', 'DnsAdmins']:
          print(f"  [!] {cn} ({members} members) *** PRIVILEGED ***")
      else:
          print(f"  {cn} ({members} members)")

  # Enumerate computers
  conn.search(base_dn, '(objectClass=computer)',
      attributes=['cn', 'operatingSystem', 'operatingSystemVersion',
                  'dNSHostName', 'userAccountControl'])
  print(f"\n[*] Computers found: {len(conn.entries)}")
  for entry in conn.entries:
      cn = entry.cn.value
      os = entry.operatingSystem.value if entry.operatingSystem else 'Unknown'
      dns = entry.dNSHostName.value if entry.dNSHostName else ''
      uac = int(entry.userAccountControl.value) if entry.userAccountControl else 0
      deleg = " [UNCONSTRAINED DELEGATION]" if uac & 0x80000 else ""
      print(f"  {cn} - {os} ({dns}){deleg}")

  # Kerberoastable accounts
  conn.search(base_dn,
      '(&(objectClass=user)(servicePrincipalName=*)(!(objectClass=computer)))',
      attributes=['sAMAccountName', 'servicePrincipalName'])
  print(f"\n[*] Kerberoastable accounts: {len(conn.entries)}")
  for entry in conn.entries:
      print(f"  {entry.sAMAccountName.value}: {entry.servicePrincipalName.values}")

  # ASREPRoastable accounts
  conn.search(base_dn,
      '(userAccountControl:1.2.840.113556.1.4.803:=4194304)',
      attributes=['sAMAccountName'])
  print(f"\n[*] ASREPRoastable accounts: {len(conn.entries)}")
  for entry in conn.entries:
      print(f"  {entry.sAMAccountName.value}")

  # Password policy
  conn.search(base_dn, '(objectClass=domainDNS)',
      attributes=['minPwdLength', 'maxPwdAge', 'minPwdAge',
                  'pwdHistoryLength', 'lockoutThreshold', 'lockoutDuration'])
  if conn.entries:
      print(f"\n[*] Password Policy:")
      for attr in conn.entries[0].entry_attributes:
          print(f"  {attr}: {conn.entries[0][attr].value}")

  conn.unbind()
  ```

### LDAP Passback Attack

::tip

```bash [Terminal]
import socketserver

class LDAPHandler(socketserver.BaseRequestHandler):
    def handle(self):
        data = self.request.recv(4096)
        print(f"[+] Connection from {self.client_address}")
        print(f"[+] Raw data: {data}")

        # Parse LDAP bind request for credentials
        # Simple bind DN and password are in the packet
        try:
            # Look for readable strings (credentials)
            readable = ''.join(chr(b) if 32 <= b < 127 else '.' for b in data)

            0x02, 0x01, 0x01,  # MessageID: 1

            0x0a, 0x01, 0x00,  # resultCode: success
            0x04, 0x00,        # matchedDN: ""
            0x04, 0x00         # diagnosticMessage: ""
```

## Pentest Checklist

::steps{level="4"}

#### Test for Anonymous Bind

Query RootDSE and attempt anonymous searches against the base DN. Anonymous bind frequently exposes the entire directory without credentials.

#### Extract RootDSE Information

Retrieve domain name, forest name, functional levels, DC hostname, and naming contexts from the RootDSE — always accessible without authentication.

#### Enumerate Users, Groups, and Computers

List all user accounts, security groups, computer objects, OUs, and trust relationships. Extract sAMAccountName, UPN, memberOf, description, mail, and UAC flags.

#### Search for Passwords in Attributes

Check description, comment, info, and extensionAttribute fields for plaintext passwords or hints — an extremely common finding.

#### Identify Kerberoastable Accounts

Find user accounts with SPNs set. Request TGS tickets and crack offline with Hashcat (mode 13100/19700/19600).

#### Identify ASREPRoastable Accounts

Find accounts with "Do not require Kerberos preauthentication" enabled. Request AS-REP and crack with Hashcat (mode 18200).

#### Extract Password Policy

Query lockout threshold, lockout duration, minimum password length, and password history to plan password spraying safely.

#### Password Spray with Discovered Users

Use kerbrute or CrackMapExec to spray common passwords against all discovered users, respecting the lockout policy.

#### Enumerate Delegation Settings

Find unconstrained delegation (full TGT forwarding), constrained delegation (S4U abuse), and RBCD opportunities.

#### Check ACL Misconfigurations

Use BloodHound or manual DACL enumeration to find GenericAll, GenericWrite, WriteDACL, WriteOwner, ForceChangePassword, and DCSync rights.

#### Attempt LDAP Relay

Check if LDAP signing and channel binding are enforced. If not, relay NTLM to LDAP for privilege escalation.

#### Check for LAPS and gMSA Passwords

Attempt to read ms-Mcs-AdmPwd (LAPS) and msDS-ManagedPassword (gMSA) attributes for local admin and service account passwords.

#### Enumerate AD CS for Certificate Abuse

Find vulnerable certificate templates (ESC1-ESC8) that allow privilege escalation to Domain Admin via certificate enrollment.

#### Collect BloodHound Data

Run bloodhound-python or SharpHound to collect comprehensive AD data for graph-based attack path analysis.

#### Document All Findings

Record evidence of anonymous bind, weak passwords, Kerberoastable accounts, ACL misconfigurations, delegation abuse, and provide remediation recommendations.
