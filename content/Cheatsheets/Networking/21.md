## Overview

This cheatsheet covers enumeration techniques for 21 of the most commonly encountered ports during penetration testing engagements. Each section includes banner grabbing, service-specific tools, NSE scripts, and manual techniques.

::tip
Always start with a full port scan before diving into service enumeration: `nmap -p- --min-rate=5000 -T4 <target>`
::

## Port 21 — FTP

File Transfer Protocol. Often misconfigured to allow anonymous access or runs outdated software with known CVEs.

### Key Checks

| Check | Command |
| ----- | ------- |
| Anonymous login | `ftp <target>` → user: `anonymous` |
| Nmap scripts | `nmap -p 21 --script ftp-anon,ftp-bounce,ftp-syst,ftp-vsftpd-backdoor <target>` |
| Banner grab | `nc -nv <target> 21` |
| Brute force | `hydra -l admin -P /usr/share/wordlists/rockyou.txt ftp://<target>` |

::code-preview
---
class: "[&>div]:*:my-0"
---
# Full FTP enumeration
nmap -p 21 --script "ftp-*" <target>

# Check for anonymous login manually
ftp <target>
> Name: anonymous
> Password: anonymous@

# List files after login
> ls -la
> binary
> get <file>

#code
```bash
# Full FTP enumeration
nmap -p 21 --script "ftp-*" <target>

# Check for anonymous login manually

> Name: anonymous
> Password: anonymous@
```

## Port 22 — SSH

Secure Shell. Enumerate supported auth methods, host keys, and attempt credential attacks when permitted.

### Key Checks

| Check | Command |
| ----- | ------- |
| Banner & version | `nc -nv <target> 22` |
| Supported auth methods | `nmap -p 22 --script ssh-auth-methods <target>` |
| Host key enum | `nmap -p 22 --script ssh-hostkey --script-args ssh_hostkey=full <target>` |
| Brute force | `hydra -l root -P /usr/share/wordlists/rockyou.txt ssh://<target>` |
| User enumeration (CVE-2018-15473) | `python3 ssh-user-enum.py --userList users.txt <target>` |

::code-preview
---

---

nmap -p 22 --script ssh2-enum-algos <target>

## Port 53 — DNS

### Key Checks

| Check | Command |
| ----- | ------- |

Server Message Block. One of the most valuable ports — enumerate shares, users, OS info, and check for known exploits.

### Key Checks

| Check | Command |
| ----- | ------- |
| Enumerate shares | `smbclient -L //<target> -N` |
| Null session check | `smbclient //<target>/share -N` |

| Nmap scripts | `nmap -p 445 --script smb-vuln-ms17-010,smb-enum-shares,smb-enum-users <target>` |
| OS info | `nmap -p 445 --script smb-os-discovery <target>` |

::code-preview
---
class: "[&>div]:*:my-0"
---
# Full SMB enumeration
enum4linux -a <target>
enum4linux-ng -A <target>

# List shares with smbclient

smbclient //<target>/share -N

# CrackMapExec for SMB recon
crackmapexec smb <target> --shares
crackmapexec smb <target> --users
crackmapexec smb <target> --pass-pol

# Check for EternalBlue (MS17-010)
nmap -p 445 --script smb-vuln-ms17-010 <target>

## Port 389 — LDAP

Lightweight Directory Access Protocol. Enumerate AD users, groups, and domain info via anonymous or authenticated binds.

### Key Checks

| Check | Command |
| ----- | ------- |
| Anonymous bind | `ldapsearch -x -H ldap://<target> -b "dc=domain,dc=com"` |
| Nmap scripts | `nmap -p 389 --script ldap-rootdse,ldap-search <target>` |
| Enumerate users | `ldapsearch -x -H ldap://<target> -b "dc=domain,dc=com" "(objectClass=person)"` |
| windapsearch | `python3 windapsearch.py -d <domain> --dc-ip <target> -U` |

::code-preview
---
class: "[&>div]:*:my-0"
---
# Anonymous LDAP query
ldapsearch -x -H ldap://<target> -b "dc=domain,dc=com" -s base

# Enumerate all users
ldapsearch -x -H ldap://<target> -b "dc=domain,dc=com" "(objectClass=user)" sAMAccountName

# Enumerate all groups
ldapsearch -x -H ldap://<target> -b "dc=domain,dc=com" "(objectClass=group)"

# Authenticated search
ldapsearch -x -H ldap://<target> -D "CN=admin,DC=domain,DC=com" -w 'Password123' -b "dc=domain,dc=com"

# windapsearch
python3 windapsearch.py --dc-ip <target> -d domain.local -u "" --users

## Port 27017 — MongoDB

MongoDB database server. Often exposed without authentication in cloud and development environments.

### Key Checks

| Check | Command |
| ----- | ------- |
| Unauthenticated access | `mongo --host <target> --port 27017` |
| Nmap scripts | `nmap -p 27017 --script mongodb-info,mongodb-databases <target>` |
| List databases | `show dbs` (in mongo shell) |
| Dump collections | `db.getCollectionNames()` (in mongo shell) |
