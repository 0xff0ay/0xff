## Service Overview

Port 514 hosts two distinct services depending on the protocol:

::card-group
  ::card{icon="i-lucide-file-text" title="UDP 514 - Syslog"}
  System logging protocol used for message logging and event notification across networks.
  
  **Key Characteristics:**
  - Centralized logging
  - No authentication by default
  - Plain text transmission
  - Used for system monitoring
  ::

  ::card{icon="i-lucide-terminal" title="TCP 514 - RSH (Remote Shell)"}
  Legacy remote shell service for executing commands on remote systems.
  
  **Key Characteristics:**
  - Trust-based authentication
  - No encryption
  - Command execution capability
  - Obsolete but still found
  ::
::

::warning
Both services are inherently insecure. Syslog transmits data in plain text, and RSH uses weak trust-based authentication. Both are prime targets for reconnaissance and exploitation.
::

::
