## Overview

Port **22** is commonly used for **SSH (Secure Shell)**, a protocol for secure remote administration. During penetration testing, enumerating SSH helps identify valid users, authentication methods, service versions, and potential weaknesses such as weak credentials or outdated software.

> SSH is widely deployed across Linux, network devices, cloud servers, and infrastructure management systems.

## Basic Port Scanning

Identify if SSH is running and determine the service version.

::code-preview
---
class: "[&>div]:*:my-0"
---
Scan port 22 and detect service version.

#code
```bash
nmap -p 22 -sV <target-ip>
```
::

Enumerate SSH users.

### Metasploit User Enumeration

---
class: "[&>div]:*:my-0"

Enumerate SSH users using Metasploit.

```bash
use auxiliary/scanner/ssh/ssh_enumusers

```

## Brute Force Login

Attempt password brute forcing against SSH.

::code-preview
---
class: "[&>div]:*:my-0"
---
Brute force SSH using Hydra.

#code
```bash
hydra -l root -P passwords.txt ssh://<target-ip>
```
::
