## How Bots Appear as Contributors

### Option 1: Free - Bot Commits (Already Working!)
The bots make commits and appear in contributions graph ✅

### Option 2: Free - Add GitHub Bots
Add these free GitHub automation bots:
- **github-rob** - Already in GitHub
- Create your own GitHub App (free)

### Option 3: Paid - Add Bot Accounts
Create free GitHub accounts → Generate tokens → Add to secrets

## Running Bots Manually

1. Go to **Actions** tab
2. Select a workflow
3. Click **Run workflow**

## Bot Features

### 🦙 Ollama Bot
- Uses local AI models (Llama3, Mistral, Codellama)
- Free and private (runs locally)
- Edit `.github/workflows/ai-ollama.yml` to change model

### 🤖 Auto Doc Bot
- Fixes formatting
- Validates links
- Updates navigation

### 🔒 Security Bot
- Scans for quality issues
- Creates issues with reports

### 📊 Weekly Stats Bot
- Tracks commit activity
- Counts new files
- Generates reports

---

**No API key needed for most bots!** Just enable workflows in Actions tab.
